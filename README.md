Dungeon-Evolution
=================

O Dungeon Evolution é um jogo de RPG de mesa criado por Maickon Rangel. Diferente dos convencionais RPGs, este visa narrar aventuras baseados em jogos eletrônicos,
trazendo consigo regras que normalmente são encontradas nos jogos de hoje. O Dungeon Evolution é um RPG independente na premiação de itens, isso quer dizer que o mestre não tem controle sobre a premiação dada aos jogadores. Tudo isso fica por conta do sistema que estou montando, onde o código fonte você pode ver aí acima.

Este código cria a simulação de encontros de itens baseado no nível do personagem. Parte das suas regras foram inspirados do atual famoso game Diablo 3 que permite o encaixe de pedras nas armas. 

O código fonte ainda não está completo. Falta muinto ainda, mas quando estiver completo estará descrito aqui.

Dungeon-Evolution 4.0
=====================

O RPG Dungeon Evolution foi atualizado para a versão 4.0 e o seu sistema também. O código por inteiro foi reescrito e será mantido em outro repositório. Nesta nova versão temos:

* Modo de login
* Salvar e editar sua ficha online
* Ficha de personagem com inventário
* Novas opções de filtro
* Tabela de itens mundanos
* Gerador de fichas base
* Bestiário com mais de 10 raças disponíveis para criação
* Sorteio de poções

As novidades são muitas com relação ao sistema anterior. Apesar do sistema estar completo, existe a possibilidade de novas funcionalidades serem adicionadas a qualquer momento.

A versão antiga foi mantida no seu endereço anterior mas a nova versão agora se encontra neste novo endereço: http://dungeonevolution.uphero.com/
